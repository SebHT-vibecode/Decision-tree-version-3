# Decision-tree-version-3

A visual decision-tree navigator for choosing measurement devices by Therapeutic Area, evidence type, and intended use.

## Files

- `index.html` — main interactive experience
- `style.css` — visual styling for the navigator
- `script.js` — decision flow and dynamic rendering logic

## Run locally

1. Open `index.html` in your browser.
2. Or use a simple local server:

```bash
cd /workspaces/Decision-tree-version-3
python3 -m http.server 8000
```

Then visit `http://127.0.0.1:8000`.
